table 태그 만들기(표만들기)

html에서 표를 구현할때

  • table 태그를 생성
  • tr태그생성(table row:행,줄) table>tr
  • td(table data):행의 내용, 열, 칸 table>tr>td
  • th(table heading):행, 열의 제목(굵은체, 가운데정렬로 태어남)table>tr>th
  • colspan="2" ==>2칸을 열병합
  • rowspan="2" ==>2칸을 행병합
